# 🕸simpleSpider
## 🕷🕷🕷基于node.js的简单爬虫
#### 🦮本仓库共有两个文件夹，simpleSpider是爬虫，还有一个是测试爬虫的网页
#### 🐎注意点：
- 🤺测试网页需要运行在服务器中，可以使用live-server将页面运行在本地服务器
- 🐏simpleSpider直接运行即可，注意端口号需要和服务器保持一致。
